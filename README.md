## Run Locally

Clone the project

```bash
  git clone https://github.com/sameeksha-purwar/Drum-Kit
```

Go to the project directory

```bash
  cd Drum kit
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
